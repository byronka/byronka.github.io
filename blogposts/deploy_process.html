The deploy process for my software

January 17, 2024

I thought it might be useful for people to see a fully-automated deploy-to-prod process.

This is a web application for a family tree / memorial. The software is written in Java, and uses the Minum web framework.

Emphasis has been put on minimalism and simplicity every step of the way, which enables continuity of productivity. I subscribe to many ideals, and wanted to incorporate those in a concrete form, as a demonstration of what the industry's software practices could become.
